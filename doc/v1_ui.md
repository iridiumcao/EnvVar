# EnvVar UI 原型说明文档（V1）

## 1. 设计目标

本 UI 设计旨在提供：

- 清晰展示环境变量信息
- 高效进行 CRUD 操作
- 提升多值变量（如 PATH）的可读性
- 降低误操作风险

设计原则：

- 简洁优先（Avoid Overdesign）
- 信息层级清晰
- 操作路径直观
- 单一页面完成主要操作

---

## 2. 整体布局

采用经典「列表 + 详情面板」结构：

```plaintext
┌────────────────────────────────────────────┐
│ 顶部工具栏                                 │
├───────────────┬────────────────────────────┤
│ 左侧变量列表   │ 右侧详情面板               │
│ （主区域）     │ （查看 / 编辑）            │
└───────────────┴────────────────────────────┘
```

布局比例建议：

- 左侧列表：40%
- 右侧详情：60%

---

## 3. 顶部工具栏（Toolbar）

### 功能项：

| 按钮 | 功能 |
|------|------|
| 刷新 | 重新加载系统环境变量 |
| 新建 | 创建新变量 |
| 展示模式切换 | 分级 / 合并 |

---

### 交互说明：

- 刷新：重新读取 Registry
- 新建：右侧面板进入“新建状态”
- 模式切换：立即刷新列表展示方式

---

## 4. 左侧变量列表（Variable List）

### 组件类型：

- DataGrid / Table

---

### 列定义：

| 列名 | 说明 |
|------|------|
| Name | 环境变量名 |
| Alias | 用户自定义名称 |
| Level | User / System |
| Preview | 值预览（截断显示） |

---

### 行行为：

- 单击：
  - 在右侧展示详情
- 当前选中行高亮

---

### 特殊标识：

对于多值变量（如 PATH）：

- 在 Name 前添加标识：
  - `[LIST] PATH`

---

### 展示模式：

#### 模式一：分级展示

- 分组显示：
  - User Variables
  - System Variables

---

#### 模式二：合并展示

- 所有变量在同一列表
- 使用标签或颜色区分 Level

---

## 5. 右侧详情面板（Detail Panel）

### 状态类型：

- 查看状态
- 编辑状态
- 新建状态

---

## 5.1 普通变量展示

字段：

- Name（只读 / 新建可编辑）
- Level（只读 / 新建可选）
- Alias（可编辑）
- Description（可编辑）
- Value（可编辑）

---

### 示例结构：

```plaintext
Name: JAVA_HOME  
Level: User  

Alias:  
[ Java Home ]

Description:  
[ JDK installation path ]

Value:  
[ C:\Program Files\Java\jdk-17 ]

[ 保存 ]   [ 删除 ]
```

---

## 5.2 多值变量展示（重点）

适用于：

- PATH
- CLASSPATH
- 其他分号分隔变量

---

### 展示结构：

```plaintext
Name: PATH  
Level: System  

Alias:  
[ System Path ]

Description:  
[ Execution paths ]

Value（原始值，可编辑）:  
[ C:\Java\bin;C:\Windows\System32 ]

----------------------------------

拆分视图（只读）：

[0] C:\Java\bin  
[1] C:\Windows\System32  
```

---

### 行为说明：

- 拆分视图自动生成
- 不支持单项编辑（V1限制）
- 修改仅作用于原始 Value 字符串

---

## 6. 新建变量流程

1. 点击「新建」按钮
2. 右侧进入新建状态：
   - Name：可输入
   - Level：选择（User / System）
   - Value：输入
   - Alias / Description：可选
3. 点击保存

---

## 7. 编辑变量流程

1. 点击左侧变量
2. 右侧加载数据
3. 修改字段
4. 点击保存

---

## 8. 删除变量流程

1. 选中变量
2. 点击删除
3. 弹出确认提示：
   - “确认删除该环境变量？”
4. 用户确认后执行删除

---

## 9. 视觉设计建议

### 风格：

- 类似 IDE（VS Code / JetBrains）
- 简洁、偏工具化

---

### 配色：

- 推荐支持深色模式
- Level 区分：
  - User：蓝色标签
  - System：橙色标签

---

### 字体：

- 等宽字体（用于 value 展示）
- 普通 UI 字体（其他内容）

---

### 间距：

- 控件间距 ≥ 8px
- 面板内边距 ≥ 12px

---

## 10. 可用性优化

- 长 value 支持横向滚动
- Preview 自动截断（带 tooltip 显示完整内容）
- 当前选中变量高亮
- 操作按钮位置固定（避免误点）

---

## 11. 错误处理（UI 层）

- 保存失败提示：
  - “写入系统环境变量失败，请检查权限”
- 删除失败提示
- 非法输入提示（空 key 等）

---

## 12. V1 限制说明

以下功能不在当前版本：

- PATH 单项编辑
- 拖拽排序
- 搜索
- 分组管理
- 快照 / 回滚

---

## 13. 后续扩展点（UI预留）

- PATH 可视化编辑器（列表操作）
- 搜索栏（顶部或左侧）
- 分组筛选
- 历史版本面板
